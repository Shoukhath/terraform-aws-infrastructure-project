# Automated Cloud Infrastructure Provisioning Using Terraform and AWS

## Project Overview
This project automates AWS infrastructure provisioning using Terraform.

## Technologies Used
- Terraform
- AWS EC2
- AWS Security Groups
- AWS CLI

## Features
- Automated EC2 instance creation
- Automated Security Group creation
- Infrastructure as Code implementation

## Project Workflow
1. Configure AWS CLI
2. Write Terraform configuration
3. Run terraform init
4. Run terraform apply
5. Verify resources in AWS Console
6. Run terraform destroy

## Outcome
Successfully automated cloud infrastructure deployment using Terraform and AWS.
